Asset folder guide

images/towers   - tower and piece images
images/monsters - monster and boss images
images/ui       - UI icons, backgrounds, effects, title screen art
audio           - sound effects and music
data            - optional balance or stage data
fonts           - title and UI font files

Use relative paths from index.html, for example:
assets/images/monsters/basic.png
